About me

Hi, I'm Gabriel Mazer, a passionate developer who created Mazzarin, a web application that showcases my skills in back-end and cloud computing. Mazzarin is a vegan restaurant website that offers traditional recipes with a modern twist.

The project was developed in Python and one of my main tasks was to create a booking API using Django REST Framework. The API allows users to make, view, and cancel reservations for the restaurant. Additionally, power users can access the DRF Browsable API and manage the menu, bookings, and users with an access token.

The whole application was deployed on an EC2 instance from AWS, which required me to learn the basics of cloud computing and overcome this challenge.

You are invited to explore this website and see what I have built. In the "Reservations" tab, you can see all the reservations in JSON format that were made from the "Book" tab.

In the "Book" tab, you can use the booking API that uses django.forms. There, you will find a user-friendly interface that prevents you from making reservations at the same time as others and lets you see all the reservations for a specific day.
