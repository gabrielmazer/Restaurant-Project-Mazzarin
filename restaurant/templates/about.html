About Us

Little Lemon is a website project created by Gabriel Mazer to demonstrate his web development skills. The site simulates a Mediterranean cuisine restaurant, with traditional recipes and a modern twist.

The project involved the creation of a reservation and booking API, using Python, Django, Django REST Framework, MySQL, Git and GitHub, JavaScript, HTML5 and CSS3.

Little Lemon is a fictional example of a restaurant and on the “Reservations” tab, you can see all the reservations in JSON format that were made from the “Book” tab.

In the "Book" tab, you can use the booking API that uses django.forms. There, you will not be able to make reservations at the same time and you can see all the reservations for that specific day.
