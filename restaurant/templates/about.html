About Us

Mazzarin is a website project created by Gabriel Mazer to demonstrate his web development skills. The site simulates a vegan restaurant, with traditional recipes and a modern twist.

The project involved the creation of a reservation and booking API, using Python, Django, Django REST Framework, MySQL, Git and GitHub, JavaScript, HTML5 and CSS3.

You are more than welcome to explore this website. In the “Reservations” tab, you can see all the reservations in JSON format that were made from the “Book” tab.

In the "Book" tab, you can use the booking API that uses django.forms. There, you will not be able to make reservations at the same time and you can see all the reservations for that specific day.
